# Templating demo

Ukázka použití balíčku pro renderování šablon ve stylu statických reactích komponent (bez stavů, bez efektů) pomocí knihovny [@czechitas/render](https://www.npmjs.com/package/@czechitas/render).

## Spuštění

```sh
npm install
```

### Vývojový server

```sh
npm start
```

### Produkční build

```sh
npm run build
```
